Compile and Execution instructions:

1. I have tested this using Python 2.7
2. Unzip the file
3. cd into python folder
4. python NaiveBayes.py ../data/imdb1     [ without flags ]
5. python NaiveBayes.py -f ../data/imbd1  [ filter stop words  ]
6. python NaiveBayes.py -b ../data/imbd1  [ Binary vectorization ]
7. Output contains average accuracy and also accuracy on each fold.
